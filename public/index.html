Ce qui se perd quand vous ne donnez pas de nouvelles.

Le vendeur perd confiance. Pas de nouvelles = il appelle une autre agence. Vous perdez le mandat que vous aviez déjà.

Zéro recommandation. Un client bien suivi vous envoie ses amis. Un client oublié ne parle jamais de vous.

Votre image souffre. Dans l'immo, la réputation se bâtit sur la relation. Un suivi négligé, ça se sait.
